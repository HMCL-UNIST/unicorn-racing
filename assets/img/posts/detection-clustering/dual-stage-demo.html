Dual-Stage Clustering — the two-boxes-merge challenge

A single left-to-right pass over the scan, exactly as detect.cpp → clustering(). Stage 1 = Adaptive Breakpoint (grow the current cluster while the gap stays under D_max = r·sin(Δφ)/sin(λ−Δφ) + 3σ). Stage 2 = on a break, re-attach the point to the nearest existing cluster if it is within a fixed new_cluster_threshold (0.4 m), else start a new one. Challenge: two 45° boxes with one beam threading the gap → Stage 2 re-associates the far box back onto the near one → two boxes become one cluster.
